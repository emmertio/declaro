Create a new @declaro/* library package following the project conventions.

Reference the shared documentation at docs/prompts/create-library.md for the full library creation guide.

Use the @declaro/core package as the reference implementation for:
- package.json structure and scripts
- tsconfig.json configuration
- Build script (scripts/build.ts)
- .gitignore
- README structure

The new library should be created in the `lib/` directory and follow all coding standards defined in docs/coding-standards.md.
